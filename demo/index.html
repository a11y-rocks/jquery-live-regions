Live Region Demos

Each of the unit test fixtures are also working demos of the plugin. If you're a screen reader user, please note that some of these test files have automatically updating Live Regions, so read each description before clicking.

To see these demos you need to clone this repo and run npm install and bower install so you have all of the necessary dependencies.

  1. Test 1: Basic Live Region with no options passed to it. Has no content. This use case is for setting up a basic empty live region that you plan on adding content to later
  2. Test 2: Basic Live Region with a few options passed to it. Has no content. This use case is for setting up a basic empty live region that you plan on adding content to later
  3. Test 3: Live region with full set of defined options. Has no content. Like 1 & 2, this use case is for setting up a basic empty live region that you plan on adding content to later
  4. Test 4: Basic Live Region with default options - This tests auto updating text. Not a real-world use case. but simulates the automatic swapping of content. Probably wouldn't perform well because of the very short interval
  5. Test 5: Basic Live Region with pre-existing text. This demonstrates the replacement of text with new text
  6. Test 5-a: Basic Live Region with pre-existing text. This demonstrates appending text to a live region
  7. Test 6: Basic Live Region with no text. This demonstrates setting up a live region you plan to add content to later (and subsequently adding the content)
  8. Test 7: This demonstrates the use of the plugin for error messages
  9. Test 8: Has no content. This demonstrates providing aria-labelledby to reference an external label
